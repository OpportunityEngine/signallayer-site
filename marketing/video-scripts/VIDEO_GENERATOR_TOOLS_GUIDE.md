## STEP-BY-STEP: PICTORY (Easiest)

### 1. Create Account
Go to pictory.ai → Sign up for free trial

### 2. Select "Script to Video"
Click "Script to Video" on the dashboard

### 3. Paste the Script
Copy the **VOICEOVER SCRIPT ONLY** section from any script file and paste it in

### 4. Choose Settings
- **Aspect Ratio**: 16:9 for YouTube, 9:16 for TikTok/Reels
- **Voice**: Select a warm, friendly AI voice
- **Style**: "Business" or "Professional"

### 5. Generate & Edit
- Pictory will auto-select stock footage
- Replace any footage that doesn't fit
- Add your logo to end card
- Add captions (highly recommended)

### 6. Export
- Export as MP4
- Upload to YouTube, social media, website

## STEP-BY-STEP: SYNTHESIA (Most Professional)

### 1. Create Account
Go to synthesia.io → Start free trial

### 2. Create New Video
Click "Create Video" → "Start from scratch"

### 3. Choose Avatar
Select a professional-looking AI presenter
- Recommend: Diverse, friendly, approachable
- Avoid: Overly corporate or stiff

### 4. Add Script
Paste the **VOICEOVER SCRIPT ONLY** section
- Break into scenes (one paragraph per scene)
- Add pauses with "..." or [pause]

### 5. Add Backgrounds
For each scene, select appropriate background:
- Office setting for problem scenes
- Software mockup for feature scenes
- Lifestyle images for emotional scenes

### 6. Add Screen Recordings
- Record your actual Revenue Radar dashboard
- Add as overlay during feature explanation

### 7. Add Music & Export

## STEP-BY-STEP: CANVA (Free Option)

### 1. Create Account
Go to canva.com → Sign up free

### 2. Create Video
Click "Create a design" → "Video" → Choose size:
- 1920x1080 for YouTube
- 1080x1920 for TikTok/Reels

### 3. Use Templates
Search "explainer video" or "business video"
- Pick a template close to your style
- Modify colors to gold (#fbbf24) and dark blue (#0a0f1a)

### 4. Add Scenes
Create one page per scene from the script
- Add text overlays
- Add stock footage/images
- Add animations

### 5. Add Audio
- Record voiceover yourself, OR
- Use Canva's AI text-to-speech, OR
- Upload ElevenLabs voiceover

### 6. Export & Share

## VOICEOVER RECORDING TIPS

### If Recording Yourself:
- Use a quiet room
- Speak slowly and clearly
- Smile while talking (it changes tone)
- Record in sections, edit together
- Use Descript or Audacity to remove "ums"

### If Using AI Voice:
- **ElevenLabs recommended voices:**
  - "Josh" - Friendly male
  - "Rachel" - Warm female
  - "Adam" - Professional male
- Add emotion markers: [excited], [concerned], [hopeful]
- Generate multiple takes, pick best

## FINAL CHECKLIST

Before publishing any video:

- [ ] Audio is clear and balanced with music
- [ ] All text is readable (not too fast)
- [ ] Logo appears at end
- [ ] Website/CTA is visible
- [ ] Captions are added (80% of social video is watched muted)
- [ ] Thumbnail is eye-catching
- [ ] Video is optimized for platform (size, length)
- [ ] First 3 seconds hook attention
