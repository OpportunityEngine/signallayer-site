## ðŸŽ¯ Quick Start

### Want a professional explainer video?
1. Open `RESTAURANT_BAR_SCRIPT.md` (or your target industry)
2. Copy the **VIDEO GENERATOR PROMPT** section
3. Paste into Pictory.ai or Synthesia.io
4. Generate video in ~10 minutes

### Want social media clips?
1. Open `SOCIAL_MEDIA_SHORT_CLIPS.md`
2. Pick a clip style (The Price Increase, The Math, etc.)
3. Copy into CapCut or Canva
4. Post to TikTok/Reels/Shorts

### Want just voiceover?
1. Open any script file
2. Copy the **VOICEOVER SCRIPT ONLY** section
3. Paste into ElevenLabs.io
4. Download MP3 in 30 seconds

## ðŸŽ¨ Brand Guidelines for Videos

### Colors
- **Primary Gold**: #fbbf24
- **Background Dark**: #0a0f1a
- **Text Light**: #e2e8f0
- **Success Green**: #22c55e
- **Alert Red**: #ef4444

### Fonts
- **Headlines**: Montserrat Bold
- **Body**: Inter Regular

### Tone
- Warm and friendly, not corporate
- Slightly humorous, not cheesy
- Empowering, not salesy
- Emotional payoff, not just features

### Logo Usage
- Always include at end card
- Minimum 3 seconds visibility
- Include website URL
